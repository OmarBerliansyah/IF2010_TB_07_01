=======================================
PANDUAN DIREKTORI KODE JAVA
=======================================

Folder: src/main/java
Fungsi: Tempat menyimpan semua file kode Java (.java)

----------------------------------------
CARA MENGATUR FILE
----------------------------------------
1. Buat folder sesuai dengan nama package:
   Contoh: Jika package 'tubes.if2010'
   Maka struktur folder:
   src/main/java/tubes/if2010/

----------------------------------------
FILE YANG BOLEH DISIMPAN:
----------------------------------------
1. File Program Utama
   - Main.java (tempat method main())

2. File Data/Model
   - Mahasiswa.java
   - Dosen.java

3. File Sistem/Logic
   - LoginSystem.java
   - Database.java

4. File Interface & Enum
   - UserInterface.java
   - JenisKelamin.java

----------------------------------------
YANG TIDAK BOLEH DISIMPAN:
----------------------------------------
 File hasil compile (.class)
 File media (gambar, suara)
 File konfigurasi
 File testing

----------------------------------------
TIPS PENTING:
----------------------------------------
1. Simpan .java sesuai packagenya
2. Beri nama file yang jelas
3. Pastikan ekstensi .java
4. Satu file = satu class utama

[!] PERHATIAN: 
Pastikan struktur package sesuai dengan 
nama folder.
